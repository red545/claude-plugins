# claude-plugins

A personal marketplace of Claude Code plugins.

## Installation

```bash
/plugin marketplace add red545/claude-plugins
```

Then install individual plugins:

```bash
/plugin install video-transcribe@nik-plugins
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [video-transcribe](./plugins/video-transcribe) | Transcribe video/audio files using OpenAI's transcription API |
